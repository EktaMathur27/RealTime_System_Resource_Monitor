# 🚀 Real-Time System Resource Monitor

A Python-based real-time system monitoring tool that tracks CPU, memory, disk, and network usage with live updates.

## 📌 Features

* 📊 Real-time CPU usage monitoring
* 💾 Memory usage tracking
* 🖥️ Disk usage statistics
* 🌐 Network activity monitoring
* 📈 Live updating dashboard (CLI-based)

## 🛠️ Tech Stack

* Python
* psutil
* time
* os

## 📂 Project Structure

```
RealTime_System_Resource_Monitor/
│── main.py
│── monitor.py
│── utils.py
│── requirements.txt
│── README.md
│── screenshots/
```

## ⚙️ Installation

1. Clone the repository:

```
git clone https://github.com/your-username/RealTime_System_Resource_Monitor.git
```

2. Navigate to the project folder:

```
cd RealTime_System_Resource_Monitor
```

3. Install dependencies:

```
pip install -r requirements.txt
```

## ▶️ Usage

Run the program:

```
python main.py
```
